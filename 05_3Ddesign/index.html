I started out by making two scans of both sides of the headphones using Revo Point Scanner, and it looked pretty good so far. However, when I tried to merge the two scans together, they were so similar that the program ended up placing both scans on the same side, so one side of the merged scan was empty. I realized this is because the headphones are so symmetrical and had no distinguishing features to single out both sides as different. So I scanned again, this time with the sticker markers.
